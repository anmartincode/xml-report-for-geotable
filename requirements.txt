# Standard Installation (No pip packages needed):

# 2. Clone this repository
# 3. Open Dynamo in Civil 3D
# 4. Load the scripts - No pip install required!

# pip install -r requirements.txt

# For standalone testing/development outside Dynamo:
# pip install lxml jsonschema

# =============================================================================
# TROUBLESHOOTING
# =============================================================================

# If you encounter import errors in Dynamo:
# 1. Verify Civil 3D API references are available
# 2. Check Dynamo Python engine is set to CPython3
# 3. Ensure Civil 3D installation is complete
# 4. Restart Civil 3D and Dynamo

# If optional packages don't install:
# 1. Update pip: python -m pip install --upgrade pip
# 2. Use virtual environment: python -m venv venv
# 3. Install individually: pip install package-name
# 4. Check Python version compatibility
